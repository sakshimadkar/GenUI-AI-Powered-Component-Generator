## GenUI – AI Component Generator for Developers

GenUI is an advanced AI-powered platform that allows developers to generate high-quality frontend UI components by simply describing their requirements in natural language.

## Features:

-AI-based component generation
-Live code editor with syntax highlighting
-Real-time preview rendering
-Copy & download generated code
-Responsive modern UI
-Dark-themed professional interface

🛠 Tech Stack:
React.js • Tailwind CSS • Monaco Editor • AI API • React Router
